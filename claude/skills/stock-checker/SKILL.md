# Stock Checker Skill
เมื่อผู้ใช้ถามราคาหุ้น ให้ทำตามนี้:
1. เรียกใช้สคริปต์ Python โดยรันคำสั่ง `python get_stock.py <TICKER>`
2. นำข้อมูล JSON ที่ได้มาสรุปและวิเคราะห์ผลให้ผู้ใช้ฟังเป็นภาษาไทย
